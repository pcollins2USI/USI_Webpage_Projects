Anatomy of Thumbnails





To have the images appear on this page instead of in a new web page requires JavaScript.
One approach is: On page load, have a blank space where the photos will appear. Use a placeholder image until you click on a thumbnail. The placeholder image could be a transparent image so that it looks like there's nothing there, or it could be an image the same color as the background.

Without JavaScript, use 'target = "_blank"' to have the images appear in a new tab.
